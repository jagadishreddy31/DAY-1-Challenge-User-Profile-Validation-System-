# DAY-1-Challenge-User-Profile-Validation-System-
Validation Rules Implemented
1. Full Name Validation
Checked whether the name contains at least two words using space count
Checked that the name does not start or end with a space
2. Email ID Validation
Checked whether the email contains @ and .
Checked that @ is not the first character
3. Mobile Number Validation
Checked whether the mobile number length is exactly 10
Checked whether it contains only digits using isdigit()
Checked that it does not start with 0
4. Age Validation
Checked whether the age is between 18 and 60
